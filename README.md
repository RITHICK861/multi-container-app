# Automated Multi-Container Application Deployment

## Technologies

- HTML
- CSS
- JavaScript
- Python Flask
- Docker
- Docker Compose
- Jenkins
- GitHub

## Project Overview

This project demonstrates an automated CI/CD workflow using Jenkins and Docker Compose.

### Features

- Responsive frontend
- Flask REST API
- Dockerized frontend and backend
- Docker Compose deployment
- Jenkins pipeline automation

## Run

```bash
docker compose up --build
```

## Stop

```bash
docker compose down
```
